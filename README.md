# 团雾识别检测系统

默认账号：

```text
admin / 123456
operator / 123456
```

启动：

```powershell
.\.venv\Scripts\python.exe init_db.py
.\.venv\Scripts\python.exe app.py
```

访问：

```text
http://127.0.0.1:5000
```

```text
图片识别
视频识别
摄像头识别
历史识别
训练统计
用户管理
```
